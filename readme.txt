Ejecucion de Porgrama:

1. Abrir la aplicacion y entrar a cualquier tabla para modificar.
2. Al entrar a una tabla se tiene varias opciones : 
	Registrar: Regitra un nuevo campo
	Consultar: Busca un campo en la tabla.
	Editar: edita cualquier valor que se quiera.
	Eliminar: elimina un campo de la tabla.
3. Para gestionar la tabla ventas Primero se debe tener el registro de un almenos un cliente y almenos un producto.
4. para consultar, Editar y eliminar se debe seleccionar a un usuario de la tabla.
5. Al estar en una tabla se puede mover entre tablas con la lista desplegable que se encuentra en la parte superior de la tabla.
6. Si se quiere volver al menu principlal se le debe dar inicio en el boton de la parte superior de la tabla.
7. Con la lista desplegable que tiene la tabla en la parte inferior se puede configurar la cantidad de productos que se quieran visualizar.

4. La Conexion de BD es con Oracle.
5. Servidor GlassFish
6. IDE NetBeans 8.2
