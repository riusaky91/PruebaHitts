Ejecucion de Porgrama:

1. Abrir la aplicacion y entrar a cualquier tabla para modificar.
2. Al entrar a una tabla se tiene varias opciones : 
	Registrar: Regitra un nuevo campo
	Consultar: Busca un campo en la tabla.
	Editar: edita cualquier valor que se quiera.
	Eliminar: elimina un campo de la tabla.
3. Para gestionar la tabla ventas Primero se debe tener el registro de un almenos un cliente y almenos un producto.

4. La Conexion de BD es con Oracle.
5. Servidor GlassFish
6. IDE NetBeans 8.2
